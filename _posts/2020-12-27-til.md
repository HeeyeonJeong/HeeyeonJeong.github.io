- Big O는 알고리즘 실행 시간의 상한을 나타낸 것 - 최악의 경우의 실행시간
  - O(n^2) 버블 정렬, 선택 정렬
  - O(n log n) 병합 정렬
  - O(n) - 선형 검색
  - O(log n) - 이진 검색
  - O(1)

<br>

- Big Ω는 알고리즘 실행 시간의 하한을 나타내는 것 - 최상의 경우의 실행시간
  - Ω(n^2) 선택 정렬
  - Ω(n log n) 병합정렬
  - Ω(n) 버블 정렬
  - Ω(log n)
  - Ω(1) - 선형 검색, 이진 검색
