- `Function Expressions`과 `Function Declarations`

<br>

- `Function Expressions`

```javascript
let 함수명 = function () {
  //구현로직
};
```

  <br>

- `Function Declarations`

```javascript
function 함수명() {
  //구현로직
}
```

  <br>

|      Function Expressions      |   Function Declarations   |
| :----------------------------: | :-----------------------: |
|  호이스팅 영향을 받지 않는다.  | 호이스팅의 영향을 받는다. |
| 클로저로 사용 or 콜백으로 사용 |                           |
